OCDS for PPPs Extension

Makes changes required by the OCDS for PPPs profile. Evaluation indicators, finance summary and project details may be moved into new extensions.

Award

The extension defines these fields in the Award object from OCDS:

Field Title Description Type(s)
evaluationIndicators Evaluation of PPP option

Key indicators that describe how the PPP option has been evaluated by government.

 EvaluationIndicators object

Contract

The extension defines these fields in the Contract object from OCDS:

Field Title Description Type(s)
financeSummary Finance summary

Key indicators that describe the financial model of the PPP.

 FinanceSummary object

EvaluationIndicators

The extension defines a new EvaluationIndicators object with these fields:

Field Title Description Type(s)
riskPremium Risk premium

The risk premium used by government when evaluating the PPP option, expressed as a decimal (for example, 0.1) and not as a percentage (for example, 10).

 number
riskPremiumDetails Risk premium details

Additional details on the risk premium used: for example, the rationale.

 string
discountRate Discount rate

The discount rate used by government when evaluating the PPP option, expressed as a decimal (for example, 0.1) and not as a percentage (for example, 10).

 number
discountRateDetails Discount rate details

Additional details on the discount rate used.

 string
netPresentValue Net present value

The discounted value of an investment's cash inflows minus the discounted value of its cash outflows.

 Value object
netPresentValueDetails Net present value details

Additional details on the calculation of the net present value.

 string

FinanceSummary

The extension defines a new FinanceSummary object with these fields:

Field Title Description Type(s)
debtEquityRatio Debt-equity ratio

The debt-equity ratio of the project, expressed as a decimal (for example, 0.1). Also called gearing or leverage.

 number
debtEquityRatioDetails Debt-equity ratio details

Additional details on the calculation of the debt equity ratio.

 string
shareCapital Share capital

The value of the capital in the project that comes from the issue of shares.

 Value object
shareCapitalDetails Share capital details

Additional details on the calculation of the share capital.

 string
subsidyRatio Subsidy ratio

The subsidy's proportion of the project value, expressed as a decimal between 0 and 1.

 number
subsidyRatioDetails Subsidy ratio details

Additional details on the calculation of the subsidy ratio.

 string
projectIRR Internal rate of return

The internal rate of return of the project, expressed as a decimal (for example, 0.1) and not as a percentage (for example, 10).

 number
projectIRRDetails Internal rate of return details

Additional details on the calculation of the project IRR.

 string
