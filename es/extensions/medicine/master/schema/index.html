Item

La extensión define estos campos en el Item objeto del OCDS:

Campo Título Descripción Tipo(s)
activeIngredients Active ingredients

An array of active ingredients, typically chemical compounds or biological substances.

 matriz de ActiveIngredient objetos
dosageForm Dosage form

The dosage form in which the medicine is available, from the dosageForm codelist.

 cadena de la lista de código dosageForm abierta
administrationRoute Administration route

The route by which the medicine can be administered, from the administrationRoute codelist.

 cadena de la lista de código administrationRoute abierta
container Container

The immediate container for the medicine. Also known as: presentation, presentation form, package.

 Container objetos

ActiveIngredient

La extensión define un nuevo ActiveIngredient objeto con estos campos:

Campo Título Descripción Tipo(s)
name Name

The name of the active ingredient. It is recommended to use a generic name from the International Nonproprietary Names (INN). Also known as: generic name, drug, substance name, active substance.

 string
strength Strength

The strength of the active ingredient. Also known as: concentration, potency.

Campo Título Descripción Tipo(s)

 cadena de la lista de código container abierta

Quantity

La extensión define un nuevo Quantity objeto con estos campos:

Campo Título Descripción Tipo(s)
unit Unit

The unit of measurement for the quantity.

 SimpleUnit objetos
value Value

The quantity, as a multiple of the unit of measurement. The quantity is expressed in interval notation, for example: "[10,10]" for 10 units, "[1,10]" for 1 to 10 units, or "[10,INF[" for 10 or more units.

 stringonumber

SimpleUnit

La extensión define un nuevo SimpleUnit objeto con estos campos:

Campo Título Descripción Tipo(s)
scheme Scheme

The codelist from which the identifier for the unit of measurement is taken, using the open unitClassificationScheme codelist. 'UNCEFACT' is recommended.

 cadena de la lista de código unitClassificationScheme abierta
id ID

The identifier for the unit of measurement, taken from the codelist referenced by the scheme field. Refer to the unitClassificationScheme codelist for details of how to find identifiers within schemes.

 string
